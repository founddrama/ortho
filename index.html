Ortho is a WordPress theme built for deployment on Rob Friesel’s F_D blog. In its present "1.0" form, the main goal is to continue its evolution toward more generic use and make it ready for general distribution. Ortho attempts to minimize page load times through minimal http requests, minified CSS and JavaScript, and optimized web graphics. Ortho also uses jQuery for special effects.

Dependencies

(1) WordPress — Ortho is a WordPress theme, so you'll need WordPress. (2) jQuery — we've included jQuery here to try and make this easy (3) jCarouselLite — a jQuery plug-in used for the Twitter marquee; again: included here

Install

Copy it into your WordPress themes directory and activate the theme.
